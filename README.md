# td
> Todo CLI application created in Go

# Usage

```shell
make build
```

To generate the binary file and 

### Create a task
```shell
./td -add My first task
```

### List all tasks
```shell
./td -list
```

### Complete the `<nth>` task
```shell
./td -complete <nth>
```

### Delete the `<nth>` task
```shell
./td -del <nth>
```
